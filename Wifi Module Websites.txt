IMPORTANT!!! For using wifi console...

1)Select Arduino-Uno-WiFi wifi node in the PC WiFi list. 

2)Next, paste into web browser...

http://192.XXX.1.XX
http://192.XXX.1.XX/home.html

IF can NOT connect try...
http://192.XXX.1.XX/home.html

3)Open the Arduino WiFi browser and connect to NETGEARXX in the WiFi Connection option.

4) Enable SLIP.
5) Set mode to STA.
6) baud rate is 9600

NOTE: Can NOT just switch power! must connect with either USB cable or barrel power connector!
-or-

http://192.XXX.XXX.1/ into browser and open the Arduino UNO WiFi console.
Then goto WiFi Settings...select NETGEARXX and enter the password to login. Now Iphone 
will see the Arduino board.

Use this setup http website access...

http://www.arduino.org/learning/getting-started/getting-started-with-arduino-uno-wifi

>>> If board can not connect to Arduino web page http://192.XXX.1.XX or any other http:// IP addresses ... press the wifi reset button (ESP B/L button). 
Then relaunch the Arduino IDE. Select the Network Arduino 192.XXX.1.XX port from the  TOOLS > PORT option and use the web address listed at the bottom.

From the web page select the WIFI setting...choose NETGEARXX and enter the password. Lastly, switch to STA mode. Now WiFi Uploading can be done.
